# DoneZo Feature Validation Prompt

You are a QA tester for the DoneZo productivity app. Your task is to validate two critical features:

## 1. Profile Name Update Feature

**Location:** Settings Page (`src/pages/Settings.jsx`)

**Validation Checklist:**
- [ ] Navigate to Settings page
- [ ] Check if "Profile Settings" section displays current user name from `donezo_settings_profile` localStorage
- [ ] Click edit button on Full Name field
- [ ] Enter new name in input field
- [ ] Click Save button
- [ ] Verify name is updated in localStorage (`localStorage.getItem('donezo_settings_profile')`)
- [ ] Verify success feedback shows (green confirmation message)
- [ ] Refresh page and confirm name persists
- [ ] Check if AuthContext user data also reflects the change

**Expected Behavior:**
- Profile name should update immediately and persist in localStorage
- Changes should be synchronized with user authentication context
- UI should show visual confirmation of successful save

---

## 2. AI Code Humanization Feature

**Location:** Task Copilot (`src/components/ui/TaskCopilotDrawer.jsx`)

**Validation Checklist:**
- [ ] Open a task and click "Copilot" button
- [ ] Enter a technical coding question (e.g., "How do I structure this in React?")
- [ ] Verify AI response is formatted as conversational text (NOT raw JSON)
- [ ] Check for markdown formatting (bold, bullet points, numbered lists)
- [ ] Verify response is concise and practical (not verbose technical jargon)
- [ ] Test multiple prompts to ensure consistent humanization
- [ ] Check error handling when API key is missing

**System Prompt Validation:**
The AI should follow this instruction:
```
"Do NOT output raw JSON. Respond conversationally, using markdown formatting 
(bolding, bullet points) to make your response easy to read. Be concise and practical."
```

**Expected Behavior:**
- AI responses are human-readable and friendly
- Technical concepts are explained in simple terms
- Responses use markdown formatting for readability
- No raw JSON or unformatted code output
- Clear, actionable advice is provided

---

## Success Criteria

✅ Profile name updates persist in localStorage  
✅ Settings page reflects all profile changes  
✅ AI responses are humanized and conversational  
✅ Markdown formatting is properly applied  
✅ Error messages are user-friendly
